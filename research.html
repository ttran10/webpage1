Main Research Subjects

  • Machine Learning and Data Mining (design machine learning based protocols and techiniques)
  • Network and Channel Coding (design and analysis of network coding based protocols for data networks)
  • Cybersecurity and Cross-domain Communication Systems (design secure protocols for wireless and cross-domain data networks)
  • Distributed and Cloud Computing (design and analysis of cloud-based/distributed systems)
